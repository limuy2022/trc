Trc命令行指令：


------------------------------------------------------------------
1.编译：trc build ***.tree
------------------------------------------------------------------
2.运行：trc run ***.ctree
------------------------------------------------------------------
3.解析ctree文件：trc dis ***.ctree/***.tree
------------------------------------------------------------------
4.输出词法单元：trc token ***.tree
------------------------------------------------------------------
5.输出语法树：trc grammar ***.tree
------------------------------------------------------------------
6.编译并执行：trc brun ***.tree
------------------------------------------------------------------
7.编译执行但不生成编译文件：trc crun ***.tree
------------------------------------------------------------------
8.清理所有ctree文件：trc clean dir
------------------------------------------------------------------
9.启动tshell：trc tshell
------------------------------------------------------------------
10.帮助：trc help
------------------------------------------------------------------
11.调试：trc tdb
------------------------------------------------------------------
