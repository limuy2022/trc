Detailed Description

内存池 对象存放在对象池中，由对象创建的特殊对象存放在公共对象池中 只有对象申请的内存才会被存放到内存池中 两者的区别在于：对象池大小固定，效率更高； 而内存池可以申请大小不固定的内存，效率稍低

必看事项： 1.本内存池底层由malloc分配内存，所以不可以支持任何跟继承扯上关系的类和结构体 （但可以分配非基础数据类型，建议仅仅用于分配基础数据类型或者C语言风格的结构体）

实现方式： 小于等于128k的用第二级分配器 大于128k的用第一级分配器

有16个free-lists。各自管理大小不同的小额区块。 将不论什么小额区块的内存需求量上调至8的倍数。如需求30，则上调至32。
