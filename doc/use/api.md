Trc对外提供的接口：

动态链接库名称               接口位置  
------------------------------------------------------------
1.Compiler                 |src/include/Compiler/Compiler.h
功能：提供编译代码的函数，可以获得每一步的调用结果
------------------------------------------------------------
2.TVM                      |src/include/TVM/TVM.h
功能：运行字节码的虚拟机
------------------------------------------------------------
3.base                     |src/include/base.h
功能：底层的基本实现，例如内存管理，异常处理之类，通常不用手动调用
------------------------------------------------------------

具体有什么函数，函数有什么作用就不放在这里了，放在头文件中自己阅读
